# Developer Portfolio — K. Rajesh

An interactive, responsive, and high-performance developer portfolio showcasing my engineering background across Full-Stack Web Development, Cloud Computing, Robotics, and IoT. 

Built with standard semantic components, utility styling layout logic, and optimized client-side interactions.

## Key Production Features

*   **Natively Managed Terminal Simulator:** An embedded sandboxed terminal execution area matching custom strings like `help`, `about`, and `skills` to dynamically inject metadata directly to the view layout.
*   **Sequential Project Filtering Arrays:** Responsive target triggers that allow viewers to filter projects dynamically (`All`, `Cloud & Deployment`, `Robotics/IoT`, `Machine Learning`) without requesting remote node updates.
*   **Global Keyboard Hotkey Interceptors:** Engineered shortcuts for technical power-users:
    *   Pressing `T` triggers smooth-scroll routing and automatically focuses on the custom interactive terminal block.
    *   Pressing `P` cycles actively through project category sequences instantly.
*   **Dynamic Client Metadata Trackers:** Includes a scroll-linked reading line progress monitor and a real-time tracking clock running native timestamp strings synchronized with Indian Standard Time (IST).
*   **Responsive Accent Shifter:** Micro-interaction utility switcher allowing developers to shift system design palette variables on the fly.

##  Stack & Engineering Tooling Definitions

*   **Frontend Engine:** HTML5, CSS3, JavaScript (ES6+), React.js, TypeScript frameworks
*   **Style Layer Layouts:** Tailwind CSS Configuration Matrix (leveraging OKLCH color spaces)
*   **Cloud Architecture & Infrastructure Deployment:** Amazon Web Services (AWS Lightsail, EC2, S3, VPC management networks)
*   **Edge Hardware & Control Automations:** C++, Arduino IDE Core, ESP32 Microcontrollers, ESP32-CAM pipelines, state machine matrices

##  Featured System Snapshots

*   **Agrica — E-Commerce Deployment (Cloud Section):** Agriculture transactional platform managed on AWS Lightsail nodes running dedicated PrestaShop instances and integrated with secure Razorpay endpoints.
*   **Robotics Architecture (Robotics/IoT Section):** Multi-joint edge robotic arm controls and quadruped servo gait tracking nodes configured with custom C++ algorithms and live web controller streams.
